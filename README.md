# Cat & Dog Classifier - Django + CNN

A full-stack machine learning project that classifies images of cats and dogs using a Convolutional Neural Network (CNN) and serves predictions through a Django web application.

# Features

CNN-based Image Classification: Classifies uploaded images as either a cat or a dog.
Real-time Web Interface: Users can upload images via a Django website and get instant predictions.
Model Storage:** Trained model is saved as `.h5` and handled efficiently using Git LFS.
Separate Training Branch: Training code is available in the `simple_code` branch for reproducibility.
